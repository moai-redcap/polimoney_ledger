# Polimoney Ledger - 運営方針

## 1. サービス概要

政治団体・選挙運動の会計担当者向けの収支管理アプリケーション。
デジタル民主主義 2030（DD2030）がホスティングし、Polimoney と連携して政治資金の透明化を推進する。

---

## 2. 料金方針

### 2.1. 寄付ベースの運営

**すべての機能を無料で提供し、運営費は寄付で賄う。**

| 項目   | 方針                           |
| ------ | ------------------------------ |
| 料金   | **無料**                       |
| 運営費 | 寄付による                     |
| 招待制 | **なし**（政治的中立性を維持） |

### 2.2. 登録制御

不正登録を防ぐため、**証明書類の提出を必須**とする。

詳細は `SPECIFICATION.md` の「登録要件」セクションを参照。

---

## 3. アクティブ台帳の制限

### 3.1. 選挙台帳

```
選挙アクティブ台帳上限 = 固定 4
  - 現在の選挙: 1
  - 次回選挙の準備: 1
  - 前回選挙の修正用: 1
  - 余裕: 1
```

### 3.2. 政治団体台帳

```
政治団体アクティブ台帳上限 = 登録政治団体数 × 2 + 1
  - 登録政治団体数 × 1（今年分）
  - 登録政治団体数 × 1（去年分の修正用）
  - 余裕: 1
```

**例:**

| 政治家タイプ | 登録団体数                        | 計算式    | 上限  |
| ------------ | --------------------------------- | --------- | ----- |
| 新人候補者   | 1（後援会のみ）                   | 1 × 2 + 1 | **3** |
| 現職議員     | 2（後援会 + 資金管理団体）        | 2 × 2 + 1 | **5** |
| 国会議員     | 3（後援会 + 資金管理 + 政党支部） | 3 × 2 + 1 | **7** |

### 3.3. 公開による監視

登録された政治団体は Polimoney で公開され、市民の目で監視できる。
不自然に多い団体登録は公の目で発見可能。

## 6. 競合分析

| サービス             | 特徴                         | 価格帯           |
| -------------------- | ---------------------------- | ---------------- |
| 弥生会計             | 汎用会計ソフト               | 年額 ¥26,000〜   |
| freee                | クラウド会計                 | 月額 ¥2,380〜    |
| **Polimoney Ledger** | 政治資金特化、Polimoney 連携 | **無料（寄付）** |

**差別化ポイント:**

- 政治資金規正法・公職選挙法に特化した科目体系
- 収支報告書の直接出力
- Polimoney とのデータ連携（透明化）
- 無料で利用可能
